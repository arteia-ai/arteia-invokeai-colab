# 📦 arteia-invokeai-colab

Usa Invoke AI en Google Colab con interfaz web completa. Sin instalación. Por ArteIA.

---

## 🧠 ¿Qué es esto?

Una forma sencilla y bonita de usar **Invoke AI** directamente desde tu navegador —  
sin necesidad de GPU ni instalaciones complicadas.

Este notebook lanza la **WebUI completa de Invoke AI** desde Google Colab.

Creado por [ArteIA](https://www.youtube.com/@arteia) para hacer la generación de imágenes con IA más accesible y divertida para todos.

---

## 🚀 Cómo se usa

1. Abre el notebook en Google Colab:

2. Sigue las instrucciones en cada bloque:

   * Preparar el entorno
   * Instalar Invoke AI y Cloudflared
   * Iniciar la interfaz

3. Espera hasta que aparezca la URL pública (algo como `https://xxx.trycloudflare.com`).

4. Ábrela en una nueva pestaña y empieza a crear imágenes.

5. Una vez tengas la interfaz en marcha, abre el Model Manager y podrás instalar SD, SDXL, SD 3.5, FLUX, LoRA, ControlNet o cualquier modelo con un solo clic.

## 🚨 IMPORTANTE 🚨

👉 Si pulsas el botón **“Ejecutar todas”** (debajo del menú), se irán ejecutando automáticamente todos los bloques del notebook, uno tras otro.  
⚙️ ¡Ideal para no tener que ir uno a uno!


> 📝 No necesitas conocimientos técnicos. Todo se ejecuta desde el navegador.
